Фильтр Заказов Доставки

Это приложение фильтрует заказы на доставку на основе ввода пользователя для 
района и времени доставки. Оно считывает заказы из файла JSON, фильтрует их в 
соответствии с заданными критериями и записывает отфильтрованные результаты в другой файл JSON.

Функции
- Проверяет ввод пользователя на правильность для района и времени доставки.
- Фильтрует заказы на основе района и заданного диапазона времени доставки.
Записывает все действия и ошибки в CSV-файл.
- Поддерживает список районов Ташкента для проверки.

Требования
- .NET SDK (версии 8.0 или выше)
- JSON-файл с заказами в формате, указанном в orders.json.

Как Запустить
- Убедитесь, что у вас установлен .NET SDK на вашем компьютере.
- Клонируйте репозиторий или загрузите исходный код.
- Поместите файл orders.json в каталог приложения.
- Откройте терминал и перейдите в каталог проекта.
- Запустите приложение с помощью команды:

    dotnet run

- Следуйте подсказкам, чтобы ввести район и время доставки.

Пример Ввода

- Район: yashnabad
- Время Доставки: 2024-10-22T15:00:00

Выходные Данные
- Отфильтрованные заказы будут сохранены в filtered_orders.json.
